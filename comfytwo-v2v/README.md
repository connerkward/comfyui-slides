A Slidev presentation — **ComfyUI Masterclass / Field Manual Vol. 01**. Covers setup, nodes, models, and practical workflows.

## Run

```bash
npm install
npm run dev
```

Opens at `http://localhost:3030`.

## Export

```bash
npm run build   # static site → dist/
```

## Stack

- [Slidev](https://sli.dev) — markdown-based presentation framework
- Custom dark theme with animated node-graph SVG backgrounds
